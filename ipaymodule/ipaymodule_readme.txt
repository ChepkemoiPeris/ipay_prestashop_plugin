#ipaymodule

#iPay module for the free prestashop e-commerce solution 

#Accept payments for your products via iPay transfer.

Navigate to your prestahop folder and open the directory shown below:

1. your_prestashop_folder/modules

2. Copy the ipaymodule to your_prestashop_folder/modules directory

Open your browser:

1. Log in to your admin section and navigate to the menu Modules. You will see a list of all the site modules. 

2. Navigate to the 'Payments and Gateways' tab and open it. You will see the module by the name 'iPay'.

3. Install the iPay module by clicking on the 'install' button against it

4. Go ahead and configure the module.
